# Aria's Code 💜

A magical toddler app made with love for Aria.

## Features

- 🎬 **Animated Reels** — Swipeable full-screen scenes (counting, animals, colors, shapes, fruits, space, music)
- 🐾 **Animal Sounds** — Tap 12 animals to hear their sounds
- 🎹 **Music Keyboard** — 8 colorful piano keys with synthesized notes
- 🫧 **Color Bubbles** — Pop floating bubbles for points
- 🔢 **Counting Game** — Tap objects to count them

## Tech

- React + Vite
- Framer Motion animations
- Web Audio API (zero sound files)
- PWA (installable)

## Run Locally

```bash
npm install
npm run dev
```
